# 안녕하세요, Rengabro입니다.

백엔드를 중심으로 서비스의 구조를 설계하고, 안정적으로 운영되는 서버 애플리케이션을 만드는 데 관심이 많은 개발자입니다.  
API 설계, 데이터 모델링, 배포 환경, 모니터링까지 서비스가 실제로 동작하고 유지되는 전체 흐름을 중요하게 생각합니다.

또한 AI를 활용해 개발 생산성을 높이고, 반복되는 작업을 자동화하며,  
LLM 기반 기능을 서비스에 자연스럽게 녹이는 방법에 관심을 가지고 있습니다.

## Tech Stack

**Backend**  
Node.js, NestJS, REST API

**Database & Cache**  
MySQL, MongoDB, Redis

**Cloud**  
AWS, GCP, Azure

**Container & Orchestration**  
Docker, Kubernetes

**IaC & DevOps**  
Terraform, Argo CD

**Monitoring**  
Grafana

**Frontend**  
Next.js, TypeScript

**Tools**  
Git, GitHub, pnpm, Turborepo, ESLint, Prettier

## 관심 있는 분야

- 백엔드 아키텍처와 안정적인 API 설계
- 데이터 모델링과 서비스 성능 개선
- AI를 활용한 개발 생산성 향상
- LLM 기반 서비스와 AI 기능 개발
- Kubernetes 기반 배포 환경과 운영 자동화
- 모니터링을 통한 서비스 안정성 개선
